# 各平台规格完整参考

## 图片尺寸

| 类型 | 平台 | 尺寸(px) | 比例 | 说明 |
|------|------|----------|------|------|
| 方形帖 | IG / LinkedIn / Facebook | 1080×1080 | 1:1 | 最通用，全平台兼容 |
| 竖版帖 | IG Feed | 1080×1350 | 4:5 | IG Feed 最佳，占屏最多 |
| 竖版故事 | IG Story / Facebook Story | 1080×1920 | 9:16 | Stories / Reels 封面 |
| 横版帖 | X / LinkedIn | 1200×675 | 16:9 | X / LinkedIn 横版 |
| 横版帖 | Facebook 链接帖 | 1200×630 | 1.91:1 | Facebook 链接预览图 |
| Carousel | IG / LinkedIn | 1080×1350 | 4:5 | 推荐，竖版占屏多 |
| YouTube 缩略图 | YouTube | 1280×720 | 16:9 | YouTube 专用 |
| TikTok 封面 | TikTok | 1080×1920 | 9:16 | 竖版全屏 |
| 头像 | 全平台 | 400×400 | 1:1 | 头像 |

## 文案规范

### Instagram
- Caption 最大：2,200 字符，推荐 ≤150 词
- Hashtag：5-7 个，放末尾用 "." 分隔开
- 首行规则：不看 more 也能读
- 帖文格式：钩子 → 空行 → 正文(空行分段) → CTA → 空行 → "." × 3 → Hashtags
- Reels：1080×1920, ≤90s
- Stories：1080×1920, ≤15s
- Carousel：最多10页

### TikTok
- Caption 最大：2,200 字符，推荐 ≤80 词
- Hashtag：3-5 个
- 视频：1080×1920, 9:16, ≤10min(推荐15-60s)
- 字幕必须内嵌（非CC）
- 封面：1080×1920

### X/Twitter
- 推文最大：280 字符
- Hashtag：1-2 个
- 图片：1200×675, 1-4张
- 视频：1920×1080, ≤2min20s
- Thread：每条独立280字符

### LinkedIn
- 帖文最大：3,000 字符，推荐 ≤1,300 字符
- Hashtag：3-5 个
- 钩子加粗 ≤50 字符
- 空行分隔每段
- 图片：1200×627
- Carousel：1080×1350, 最多300页(PDF)
- 视频：1920×1080, ≤10min

### YouTube
- 标题最大：70 字符（含关键词）
- 描述最大：5,000 字符，推荐前150词放核心信息
- Hashtag：3-5 个，在描述区
- 缩略图：1280×720
- 视频：1920×1080(横版) / 1080×1920(Shorts)
- Shorts：≤60s
- 时间戳：视频≥3min时添加

### Facebook
- 帖文推荐：≤250 词
- Hashtag：3-5 个
- 图片：1200×630
- 视频：推荐 1:1 或 4:5, ≤240min
- Stories：1080×1920, ≤20s(广告) / 60s(有机)
